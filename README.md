# Student Management System (Console Application)

## Overview
The **Student Management System** is a simple yet functional console application built with **C#** and **.NET 8**. It allows users to manage student records in a clean and organized manner.

---

## Features
- **Add Student:** Input student details and store them in the system.
- **View Students:** Display a list of all students.
- **Basic Validation:** Ensures that student data is entered correctly.

---

## Technologies
- **C#**
- **.NET 8**
- **Console Application**

---

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes & Constructors
- Collections (`List`)
- Exception Handling
